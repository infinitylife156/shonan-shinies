# 画像配置ガイド / Image Placement Guide

このウェブサイトで使用する画像は、`images` フォルダの中に配置してください。
以下のファイル名で画像を保存すると、自動的にウェブサイトに反映されます（または、コード内のファイル名をこれらに合わせて変更します）。

Please place the images for the website in the `images` folder.
Save the images with the following filenames to have them appear on the website.

## 必要な画像 / Required Images

1.  **メインビジュアル (ヒーロー画像) / Hero Image**
    *   ファイル名: `hero.jpg`
    *   説明: トップページの最初に表示される大きな画像です。子供たちがバレーボールを楽しんでいる、明るく元気な写真が良いです。
    *   推奨サイズ: 横幅 1920px 程度

2.  **「バレーボールを楽しもう」セクション / "Enjoy Volleyball" Section**
    *   ファイル名: `enjoy.jpg`
    *   説明: 楽しさを伝える写真。笑顔のアップや、ハイタッチしているシーンなど。
    *   推奨サイズ: 横幅 800px 程度

3.  **練習風景 / Practice Scene**
    *   ファイル名: `practice.jpg`
    *   説明: 練習中の様子。
    *   推奨サイズ: 横幅 800px 程度

4.  **ロゴ / Logo** (もしあれば / If available)
    *   ファイル名: `logo.png`
    *   説明: チームのロゴ。背景が透明なPNG形式が望ましいです。

## 注意点 / Notes

*   画像ファイル名は半角英数字にしてください。
*   `.jpg` または `.png` 形式を推奨します。
*   写真の向き（横長、縦長）によってレイアウトが少し変わる可能性がありますが、基本的には横長の写真が使いやすいです。
